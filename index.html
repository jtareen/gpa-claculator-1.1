How to use

Welcome to our GPA Calculator! Follow these simple steps to calculate your GPA and CGPA:

  1. Enter Your Previous CGPA

    • Previous CGPA: Input your previous cumulative grade point average (CGPA).
    • Credit Hours: Enter the total number of credit hours you've completed before the current semester.
    • If you only want to calculate your GPA for the current semester, you can leave these fields empty.

  2. Input Your Current Semester Grades

    3. For each course you're taking this semester:

    • Course Name: Input the name or code of the course (optional).
    • Credit Hours: Enter the credit hours associated with each course.
    • Grade: Select the grade you received for the course from the dropdown menu. The corresponding grade points will automatically populate based on the grading system.

  3. Add More Courses (if needed)

    • Click on the "Add More Courses" button if you have more than seven courses to input. Additional fields will appear.

  4. Calculate Your GPA and CGPA

    • Once all the necessary information is entered, click the "Calculate" button.
    • Your GPA for the current semester and updated CGPA will be displayed below the table.

  5. Reset Form

    • If you need to start over, click the "Reset" button to clear all fields and input new data.
